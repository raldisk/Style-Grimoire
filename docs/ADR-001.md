# ADR-001: Deployment Architecture and API Interaction Strategy

**Status:** Implemented  
**Date:** 2026-03-08

---

## Context

Style Grimoire interacts with the Anthropic Messages API to generate style outputs. Two deployment contexts exist with different requirements.

---

## Decision

### Context A — Claude Artifact Sandbox

Direct API call from the client. Auth headers are injected by the claude.ai runtime automatically.

```
React UI → Anthropic Messages API
```

`import.meta.env.PROD` is `false` → `API_URL = "https://api.anthropic.com/v1/messages"`

---

### Context B — Self-Hosted (Implemented)

All API calls route through an Express proxy. The API key lives exclusively on the server.

```
React UI → /api/generate → Express (server/index.js) → Anthropic Messages API
```

`import.meta.env.PROD` is `true` → `API_URL = "/api/generate"`

**Implementation:**
- `server/index.js` — Express proxy, injects `x-api-key` + `anthropic-version` headers
- `ANTHROPIC_API_KEY` set via environment variable at runtime — never in browser bundle
- Dockerfile builds frontend → copies into Express server → single container on port 3000
- Vite dev proxy: `/api` → `localhost:3000` during local development

---

## Security Properties

| Concern | Sandbox | Self-Hosted |
|---------|---------|-------------|
| API key exposure | None (runtime-injected) | None (server-side only) |
| Key in browser bundle | No | No |
| Key in Docker image | N/A | No (runtime env var) |

---

## Running Self-Hosted

**Docker (recommended):**
```bash
docker build -t style-grimoire .
docker run -p 3000:3000 -e ANTHROPIC_API_KEY=sk-ant-... style-grimoire
```

**Local dev:**
```bash
node server/index.js          # port 3000
npm run dev                   # port 5173 (proxies /api → 3000)
```

---

## Future Considerations

- **Retry logic** — currently client-side in `retryFetch()`. Could move server-side for shared backoff across users.
- **Rate limiting** — add per-IP limits in Express middleware.
- **SSE streaming** — switch to `stream: true` + Server-Sent Events for lower perceived latency.
- **Zod validation** — validate request payloads in the proxy before forwarding.
